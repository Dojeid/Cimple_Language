# Cimple Language
For more info check main branch

This milestone marks the transition from a syntax-focused experiment to a functional language frontend.

## Design Goals

- Clean Python-style syntax
- Predictable performance
- **LLVM-based backend** (In Progress)
- Low-level control without syntactic noise
- Minimal core language, extensible by design

## Roadmap

The next major focus areas:

- Full LLVM IR generation for native execution
- Advanced Type Inference system
- Standard Library (Cimple StdLib)
- GPU programming extensions
- Cross-platform support (Linux/macOS)

Cimple is currently in active development. Expect rapid iterations.
